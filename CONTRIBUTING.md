# Contributing

Thanks for helping improve this template.

## Local workflow

1. Install dependencies.

```bash
npm install
```

1. Run strict type checks.

```bash
npm run typecheck
```

1. Build TypeScript outputs.

```bash
npm run build
```

## Contribution guidelines

- Keep TypeScript files as source of truth.
- Do not manually edit generated JavaScript outputs.
- Prefer small, focused pull requests.
- Update docs when behavior changes.

## Commit guidance

- Use clear, descriptive commit messages.
- Keep unrelated changes out of the same commit.

## Pull request checklist

- Build passes locally.
- Typecheck passes locally.
- README and setup docs remain accurate.
